## Installation
untuk menjalankan docker compose ini, anda harus menginstall docker dan docker-compose terlebih dahulu.
perintah untuk menjalankan docker compose ini adalah:
```bash
docker-compose up -d
```
untuk menghentikan docker compose ini adalah:
```bash
docker-compose down
```
